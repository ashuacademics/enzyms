LCMS Pipeline Information

Pipeline Overview

The LCMS (Liquid Chromatography-Mass Spectrometry) pipeline processes and analyzes mass spectrometry data, particularly from LCMS-QTOF instruments. The pipeline is designed to detect and align features across multiple samples, enabling researchers to identify and quantify compounds in complex biological mixtures.

Inputs Required

The pipeline requires the following input files:

  • SMILES File: A file containing the SMILES strings representing the chemical structure of the substrates.
  • Parameters File: A file with parameters for feature detection and alignment, including mass tolerance, retention time windows, etc.
  • Samples File: A CSV file listing the samples and their corresponding enzymes or treatments, formatted as:
Enzyme,Sample
CPD041_EVC1,evc
CPD041_EVC2,evc
CPD041_EVC3,evc
CPD041_KGD001,sample
CPD041_KGD002,sample
CPD041_KGD003,sample
CPD041_KGD005,sample
CPD041_KGD007,sample
CPD041_KGD008,sample
  • mzML Files: Mass spectrometry data files in the mzML format, converted from the raw .d format using ProteoWizard. Download ProteoWizard
